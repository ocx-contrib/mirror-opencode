# opencode

opencode is an open source AI coding agent built for the terminal. It pairs a
fast TUI with a headless server, works with any model provider, and runs
agentic coding sessions, MCP servers, and the Agent Client Protocol from a
single self-contained binary.

## What's included

- **opencode** — terminal AI coding agent: TUI, headless server, `run`, agents, MCP, and provider management
